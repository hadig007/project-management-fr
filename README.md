# project_management

Using:
- Frontend: VueJs v3
- Backend: Golang v1.19

## Run
``` bash
# clone respository
git clone git@github.com:hadigun007/project-management.git

# enter folder
cd project-management

# run development
npm run serve
```
## Build

``` bash 
# generate code to dist folder
npm run build
```
